# cls-better
i got tired of typing in clear so i decided to make cls so i didn't have to type in an extra 2 letters



# installing cls

type this shell script into your terminal to install cls-better quickly.
note: cls-better requires gcc, make, and git to install.

<code>curl -fSSl https://raw.githubusercontent.com/0xTex5/cls-better/refs/heads/main/install.sh | sudo bash</code>


# building cls

make sure you have gcc, and type

<code>make</code>

into the console

boom.



# installing cls from source

type

<code>make install</code>

into the console

it will automatically build a new version and install it to your PATH.
